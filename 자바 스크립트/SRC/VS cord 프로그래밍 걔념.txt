  HTML이란?
- 웹 브라우저에서 문서 및 웹 페이지가 표시되는 방법을 규정하는 언어이며
- 웹 사이트의 구조를 만들 수 있음

  CSS이란?
- HTML로 만들어진 문서의 스타일을 지정하는 스타일 시트 언어(유명함)
- 간단한 애니메이션 효과를 만들 수 있음
- 주로 정적인 시각효과를 만드는데 사용

   Java Script이란?
- 웹페이지에 포함되어 있는 다양한 구성요소들의 동작을 변경할 수 있는 도구를 제공 
- 데이터 기반의 언어
- 브라우저에 불러와서 사용할 수 있음

Java Script 기본 명령어 
shift + ait + f  : 정렬
shift + ait + 아래 화살표 : 복사
Ctrl + / : 주석처리

기본 구성 설명

<!DOCTYPE html> : 이 문서가 HTML5 문서임을 선언
<html lang="ko"> : HTML문서의 시작을 나타내며 lang 속성을 이용하여 문서의 기본언어가 한국어임을 명시
<head> : 문서의 메타데이터와 제목 등을 포함하는 머리말
  <meta charset="UTF-8"> : 문서의 문자 인코딩을 UTF - 8 으로 설정
  <meta name="viewport" content="width=device-width, initial-scale=1.0"> : 반응형 웹 디자인을 위한 뷰포트 설정
  width=device-width : 뷰포트의 너비를 디바이스의 너비만큼 설정
  initial-scale=1.0 : 페이지가 처음 로드될 때 기본 확대/축소 수준을 지정
  <title>Document</title> 
  <body>: 뷰포트의 너비를 디바이스의 너비만큼 설정
